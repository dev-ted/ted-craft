**ted-craft** is a marketplace of skills and helpers for Cursor, Claude, and Codex — browse what each one does, then add it to your setup.

## Quick start

Ask your agent to route you to something useful:

<CliCommand>npx ted-craft start</CliCommand>

Or install a skill directly (Cursor example):

<CliCommand>npx ted-craft add click-up-maintainer -a cursor -g -y</CliCommand>

Use `-a claude` or `-a codex` if you work in those agents instead.
