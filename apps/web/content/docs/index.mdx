**ted-craft** is a registry + docs site + CLI for sharing agent artifacts.

## Quick start

<CliCommand>npx ted-craft start</CliCommand>

Then install something useful:

<CliCommand>npx ted-craft add click-up-maintainer -a cursor -g -y</CliCommand>
